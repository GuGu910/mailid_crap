# mailid_crap

	    The python script which is running on selenium will open the chrome browser and searching(both 
	    google and yahoo) the content by the string `(Ex. "domain" email)` and crap the whole <html> 
	    content as string.
	    
	    Process the whole string content and matching only mail ID's.
	    
	    
	    ## Pre-requesites:
	    
	    1. Python 3.6 or higher than this version should be installed.
	    2. Selenium should be installed.
	    
	    
	    ## How to run:
	    
	    1. Clone the code by `git clone https://github.com/GuGu910/mailid_crap.git`
	    2. Put a punch of domains in a file `domains`.
	    3. Just run a `mail_crap.py`.
	    4. Finally all mails will be saved in a file mails.csv
